## 🎓 COMO USAR ESTE NOVO SETUP

### Para Desenvolvedores
```bash
# Consultar documentação
cat ARCHITECTURE.md    # Entender organização
cat CONVENTIONS.md     # Seguir padrões de código
cat MAINTENANCE.md     # Executar tarefas comuns

# Adicionar novo link ao menu
vim assets/js/menu-links.js
# Editar o array menuLinks

# Criar nova página
# 1. Copiar template de HTML existente
# 2. Adicionar link em assets/js/menu-links.js
# 3. Incluir <nav data-shared-menu></nav>
# 4. Incluir <script src="assets/js/menu-links.js">
```

### Para Gestores/Não-técnicos
```
Quer adicionar novo link ao menu?
→ Pedir: "Adicione novo-mapa.html ao menu"
→ Técnico edita: assets/js/menu-links.js (1 linha)
→ Pronto! Menu atualiza em TODAS as 7 páginas
```
