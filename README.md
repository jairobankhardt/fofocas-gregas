# 🎭 Fofocas Gregas

> _Onde os mitos da Grécia Antiga viram fofoca — e toda fofoca esconde uma tragédia._

Repositório de pesquisa dramatúrgica para o espetáculo teatral **Fofocas Gregas**, contendo um catálogo completo das obras do teatro grego antigo e mapas interativos de relações entre personagens.

---

## 📖 Sobre o Projeto

O espetáculo _Fofocas Gregas_ explora as conexões entre os personagens das tragédias e comédias gregas antigas — personagens que habitam múltiplas obras, transitam entre famílias e ciclos míticos, e cujas histórias se entrelaçam de formas surpreendentes.

Este repositório reúne o material de pesquisa em formato de páginas HTML interativas, publicadas via GitHub Pages.

## 🗺️ Mapas de Relações

Cada mapa é uma página HTML interativa — sem dependências externas, abre direto no navegador.

### Como usar

- **Clique** em qualquer personagem para ver sua descrição e em quais obras aparece
- **Arraste** para navegar pelo mapa
- **Scroll** para aproximar ou afastar
- Botão **Rótulos** para mostrar/ocultar nomes
- Botão **⟳ Resetar** para voltar à visão inicial

### Legenda geral de conexões

| Cor da seta         | Tipo de relação                     |

## 🏛️ Contexto Histórico

As peças catalogadas foram encenadas nos festivais dionisíacos de Atenas — as **Grandes Dionísias** (março) e as **Leneanas** (janeiro) — durante o Século de Ouro ateniense (séc. V a.C.). Das centenas de peças escritas, sobreviveram apenas 48 completas:

- **Ésquilo** (525–456 a.C.): 7 de ~90 peças
- **Sófocles** (496–406 a.C.): 7 de ~120 peças
- **Eurípides** (480–406 a.C.): 19 de ~92 peças
- **Aristófanes** (~446–386 a.C.): 11 de ~40 peças
- **Menandro** (342–290 a.C.): 4+ de ~100 peças
